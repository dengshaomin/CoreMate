# AndroidMate

[![](https://jitpack.io/v/DrkCore/CoreMate.svg)](https://jitpack.io/#DrkCore/CoreMate)

在工程的build.gradle中添加

```
allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
}
```

在module的build.gradle添加

```
	dependencies {
	        compile 'com.github.DrkCore:CoreMate:1.0.0-beta'
	}
```
